Uo# FastAPI Video data catalog

## Develop

### Setup:

Right click `v-data-catalog` -> Mark directory as -> Sources root

### Run

Go to workdir:
```shell
cd v-data-catalog
```

Run dev server:
```shell
fastapi dev
```
